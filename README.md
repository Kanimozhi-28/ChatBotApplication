# 🤖 ChatBot Application

A modern, full-stack HR Chatbot system designed for seamless employee interactions. This project integrates a high-performance **FastAPI** backend with a responsive **Next.js** frontend, providing a robust solution for automated HR support.

---

## 🚀 Overview

The ChatBot Application is built to handle common HR queries, manage employee data, and provide real-time responses. It uses a structured database and a secure API layer to ensure data integrity and user privacy.

---

## 📂 Project Structure

- **backend/**: FastAPI server handling chatbot logic, database interactions, and real-time communication.
- **hr-chat-app/**: Modern Next.js frontend with Tailwind CSS for a premium user experience.
- **Instructions.md**: Comprehensive guide for local setup and deployment.

---

## 🛠️ Tech Stack

### Backend
- **Framework:** [FastAPI](https://fastapi.tiangolo.com/)
- **Server:** [Uvicorn](https://www.uvicorn.org/)
- **Real-time:** [Socket.io](https://python-socketio.readthedocs.io/)
- **Security:** bcrypt, PyJWT
- **Database:** SQLite (SQLAlchemy)

### Frontend
- **Framework:** [Next.js (App Router)](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **State Management:** React Hook Form
- **Communication:** Axios, Socket.io-client
