# ChatBot Application

A comprehensive HR Chatbot application featuring a Next.js frontend and a FastAPI backend.

## Project Structure

- `backend/`: FastAPI server handling chatbot logic and database interactions.
- `hr-chat-app/`: Next.js frontend for the user interface.
- `Instructions.md`: Detailed instructions for setting up and running the application.

## Getting Started

Refer to the `Instructions.md` file in the root directory for setup and execution details.
